Important FAQ
What is Document Object Model(DOM)?

Document Object Model or in short DOM is a structured representation of a HTML document which gives the power to access all its element through Java Script as they are objects.

How do you select an element from the DOM?

To select an element from the DOM, you can use document.getElementById('id') to select by ID or document.querySelector('selector') to select the first element that matches a CSS selector. These methods allow JavaScript to interact with and manipulate specific elements in the document.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element to manage events on its child elements. It's useful because it improves performance and handles dynamically added elements without needing multiple event listeners.

How do you manipulate an elements attributes and styles using the DOM?

You can manipulate an element's attributes using setAttribute() and getAttribute(), and change its styles by accessing the style property (e.g., element.style.color = 'red';). This allows you to dynamically update the element's properties and appearance.
